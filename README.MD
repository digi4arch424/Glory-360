# Glory360 — Panoramic Virtual Creator

Glory360 is a simple panoramic virtual tour creator and viewer currently in development.

The project focuses on lightweight 360° experiences with:

- Panorama viewing
- Hotspot placement
- Scene linking
- Edit and View modes
- Browser-based virtual walkthroughs

Future expansion may include:

- VR support
- Multiplayer walkthroughs
- Mobile support
- Cloud tour storage
- Interactive maps
- AI-assisted scene generation

https://digi4arch424.github.io/Glory-360/
